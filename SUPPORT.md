# TransferX Support

Thank you for purchasing **TransferX**.

We appreciate your support and hope the project helps accelerate your development.

---

# Support Includes

The following support is available:

- Installation assistance
- Configuration guidance
- Bug fixes for reported issues
- Clarification of existing features
- Documentation assistance
- Minor issue troubleshooting

---

# Support Does Not Include

Support does **not** include:

- Custom feature development
- Complete redesigns
- Third-party service configuration
- Server administration
- Custom integrations
- Major code modifications
- Database migration services

These services may be offered separately if available.

---

# Before Contacting Support

Please ensure that you have:

- Read the README.md
- Followed INSTALL.md
- Configured the `.env` file correctly
- Installed all required dependencies
- Checked the troubleshooting section
- Reviewed any existing documentation

Many common issues can be resolved by following the setup instructions carefully.

---

# Information to Include

When requesting support, please provide:

- Project version
- Python version
- Operating System
- Browser (if applicable)
- Error message
- Steps to reproduce
- Screenshots (if applicable)
- Relevant logs (excluding sensitive information)

Providing complete information helps us resolve issues more quickly.

---

# Bug Reports

If you discover a bug, include:

- Description
- Expected behavior
- Actual behavior
- Steps to reproduce
- Screenshots
- Console logs (if available)

---

# Feature Requests

Feature suggestions are always welcome.

Please include:

- Feature description
- Use case
- Expected workflow
- Benefits of the feature

Feature requests may be considered for future releases but are not guaranteed.
